# Font Setup

The TUI works with any monospace font, but **JetBrains Mono Nerd Font** enables
the full icon set and is bundled in the `font/` directory.

## Option 1 — Install the bundled font

1. Open the `font/` folder in the project root.
2. Right-click `JetBrainsMonoNerdFont-Regular.ttf` → **Install for all users**.

## Option 2 — Download from the website

Visit https://www.nerdfonts.com/font-downloads, search for
**JetBrainsMono**, download the zip, extract it, and install the `.ttf` files
the same way.

## Configure your terminal

| Terminal | Path |
|----------|------|
| **Windows Terminal** | Settings → Profile → Appearance → Font face → `JetBrainsMono Nerd Font Mono` |
| **VS Code integrated terminal** | `terminal.integrated.fontFamily`: `"JetBrainsMono Nerd Font Mono"` |
| **Other terminals** | Set the font in the terminal's own preferences to the installed font name |

Restart the terminal after changing the font.
